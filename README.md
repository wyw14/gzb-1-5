# 🌱 家庭绿植养护智能管家

一个功能完整的家庭绿植养护管理系统，帮助您科学管理家中的植物。

## ✨ 功能特性

### 1. 植物管理
- 录入植物品种、光照/湿度偏好、浇水施肥周期
- 支持12种常见植物快捷选择（绿萝、多肉、吊兰、发财树等）
- 按养护难度（简单/中等/困难）筛选
- 植物信息编辑、删除功能

### 2. 智能养护提醒
- 自动计算下次浇水、施肥时间
- 养护任务分类展示（已过期/今天/即将到来）
- 浏览器通知推送（需开启通知权限）
- 一分钟自动检查养护任务

### 3. 成长记录
- 上传植物生长照片
- 时间轴展示成长历程
- 照片对比功能，直观对比植物生长变化
- 支持添加照片备注

### 4. 病虫害识别
- 内置10种常见病虫害知识库
- 前端关键词智能匹配
- 常见症状快捷搜索
- 包含症状描述、治疗方法、预防措施

### 5. 数据统计
- 存活率统计面板
- 养护达标率计算
- 月度养护趋势图表
- 难度分布饼图
- 需要关注的植物提醒

### 6. 年度报告
- 按年度生成养护报告
- 年度数据汇总统计
- 植物养护详情
- 导出JSON格式报告

## 🛠️ 技术栈

- **前端**: Vue 3 + Element Plus
- **后端**: Express.js
- **数据存储**: JSON文件
- **图表**: Chart.js
- **通知**: 浏览器Notification API

## 📡 API 接口

### 植物管理
- `GET /api/plants` - 获取植物列表
- `POST /api/plants` - 添加植物
- `PUT /api/plants/:id` - 更新植物
- `DELETE /api/plants/:id` - 删除植物
- `POST /api/plants/:id/water` - 记录浇水
- `POST /api/plants/:id/fertilize` - 记录施肥

### 照片管理
- `GET /api/plants/:id/photos` - 获取植物照片
- `POST /api/plants/:id/photos` - 上传照片
- `DELETE /api/photos/:id` - 删除照片

### 病虫害识别
- `GET /api/pests` - 获取病虫害列表
- `GET /api/pests?keyword=xxx` - 关键词搜索
- `GET /api/pests/:id` - 获取病虫害详情

### 统计与报告
- `GET /api/statistics` - 获取统计数据
- `GET /api/notifications` - 获取养护提醒
- `GET /api/report/yearly?year=2024` - 获取/导出年度报告

## 🎯 使用说明

### 添加植物
1. 点击"添加植物"按钮
2. 填写植物信息，或从快捷选择中选择常见植物
3. 设置浇水和施肥周期
4. 保存后系统自动计算下次养护时间

### 养护提醒
1. 在侧边栏点击"🔔 开启通知"按钮
2. 允许浏览器通知权限
3. 系统会自动推送养护提醒

### 成长记录
1. 在植物卡片点击"更多" → "照片"
2. 上传照片并添加备注
3. 在"成长记录"页面查看时间轴
4. 开启"对比模式"选择两张照片进行对比

### 病虫害识别
1. 在"病虫害识别"页面输入症状关键词
2. 或点击常见症状标签快速搜索
3. 查看匹配结果，包含症状、治疗方法、预防措施

### 年度报告
1. 在"年度报告"页面选择年份
2. 查看年度养护总结
3. 点击"导出JSON"下载报告文件

## 💡 小贴士

- 首次使用请开启浏览器通知权限，以便及时接收养护提醒
- 定期上传植物照片，记录成长历程
- 发现植物异常时，及时使用病虫害识别功能
- 建议每月查看数据统计，了解养护情况
- 年度结束时导出报告，总结养护经验

## 📝 注意事项

- 数据存储在本地JSON文件中，建议定期备份
- 照片存储在 public/uploads 目录下
- 通知功能需要浏览器支持Notification API
- 建议使用Chrome、Edge、Firefox等现代浏览器
